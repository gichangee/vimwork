# 문제 1
vimgolf put 5f0f5fbe280fbf000c233304

* #### 나의 점수 : 8 점
* #### 나의 정답
  * GWi"< END >< C-@ >ZZ
* #### 해설
  
  대문자 G는 파일의 마지막 행의 맨 앞으로 이동
  
  대문자 W는 문자 단위로 앞으로 이동한다. 
  
  소문자 i는 현재 커서 위치에서 입력 모드를 시작한다.
  
  END는 현재 라인 끝으로 이동한다.
  
  C-@(ctrl+@)는 앞에서 입력한 것들을 그대로 입력한 후 일반모드로 다시 돌아간다.
  
  대문자 ZZ는 변경사항 저장 후 종료이다.
